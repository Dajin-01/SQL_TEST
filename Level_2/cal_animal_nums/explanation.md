# [SQL] 동물 수 구하기

## 문제 설명

`ANIMAL_INS` 테이블에는 동물 보호소에 들어온 동물들의 정보가 저장되어 있다.

이 문제는 **동물 보호소에 들어온 동물이 총 몇 마리인지 조회하는 문제**이다.

동물 한 마리의 정보가 `ANIMAL_INS` 테이블에 한 행으로 저장되어 있으므로,
테이블의 전체 행 개수를 세면 보호소에 들어온 동물 수를 구할 수 있다.

---

## 테이블 구조

### ANIMAL_INS

| Column name      | Type       | Nullable | 설명          |
| ---------------- | ---------- | -------- | ----------- |
| ANIMAL_ID        | VARCHAR(N) | FALSE    | 동물의 아이디     |
| ANIMAL_TYPE      | VARCHAR(N) | FALSE    | 생물 종        |
| DATETIME         | DATETIME   | FALSE    | 보호 시작일      |
| INTAKE_CONDITION | VARCHAR(N) | FALSE    | 보호 시작 시 상태  |
| NAME             | VARCHAR(N) | TRUE     | 동물의 이름      |
| SEX_UPON_INTAKE  | VARCHAR(N) | FALSE    | 성별 및 중성화 여부 |

---

## 문제 조건

* `ANIMAL_INS` 테이블에서 데이터를 조회한다.
* 보호소에 들어온 동물의 전체 수를 구한다.
* 컬럼 이름은 문제에서 따로 지정하지 않았으므로 별칭은 필수는 아니다.

---

## 풀이 설명

이 문제는 **COUNT 집계 함수를 사용하는 문제**이다.

`ANIMAL_INS` 테이블에서 동물 한 마리는 하나의 행으로 저장되어 있다.

따라서 전체 동물 수를 구하려면 테이블의 전체 행 개수를 세면 된다.

```sql
COUNT(*)
```

`COUNT(*)`는 테이블에 존재하는 전체 행의 개수를 계산한다.

따라서 `ANIMAL_INS` 테이블 전체를 대상으로 `COUNT(*)`를 사용하면 보호소에 들어온 동물 수를 구할 수 있다.

---

## 정답 코드

```sql
SELECT COUNT(*)
FROM ANIMAL_INS;
```

---

## 핵심 개념 정리

| SQL 문법            | 역할                    |
| ----------------- | --------------------- |
| `COUNT(*)`        | 테이블의 전체 행 개수를 계산      |
| `FROM ANIMAL_INS` | 동물 보호소 입소 정보 테이블에서 조회 |

---

## 정리

이 문제의 핵심은 **동물 한 마리의 정보가 테이블의 한 행으로 저장되어 있다는 점**이다.

따라서 `COUNT(*)`를 사용해 `ANIMAL_INS` 테이블의 전체 행 개수를 구하면 된다.
